# 🤖 Jarvis AI Voice Assistant (Python)

## 📌 Overview

Jarvis is a voice-controlled AI assistant built using Python that can perform various tasks such as web browsing, information retrieval, music playback, and email automation using voice commands.

---

## 🎯 Features

* 🎙️ Voice command recognition
* 🔊 Text-to-speech response
* 🌐 Open websites (Google, YouTube, LinkedIn)
* 📚 Wikipedia search and summaries
* 🎵 Music playback
* ⏰ Real-time system time
* 📧 Send emails via voice command

---

## 🛠️ Tech Stack

* Python
* SpeechRecognition
* pyttsx3 (Text-to-Speech)
* Wikipedia API
* smtplib
* PyAudio

---

## 🔄 How It Works

1. Takes voice input using microphone
2. Converts speech to text
3. Matches command with predefined tasks
4. Executes the task and responds via voice
